# Employee Management System (Java)

## Project Description
This is a **console-based Employee Management System** developed using **Core Java**.  
The project allows users to **Add, View, and Delete employee records** using arrays without using ArrayList or databases.

This mini project is designed to strengthen understanding of:
- Java basics
- Classes & Objects
- Arrays
- Loops
- Conditional statements
- Menu-driven programs

---

## Features
-  Add Employee Details  
-  View All Employees  
-  Delete Employee by ID  
-  Menu-driven console application  

---

## Technologies Used
- **Language:** Java  
- **Concepts:**  
  - OOP (Classes & Objects)  
  - Arrays  
  - Loops  
  - Scanner Class  
  - Control Statements
